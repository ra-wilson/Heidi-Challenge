# Kinetic Share Prototype

Next.js prototype for a reciprocal patient-data sharing network demo.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Zustand global state

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Demo controls

Use the top-right controls to switch:

- Membership tier: Out / Basic / Standard / Full
- Sharing enabled toggle
- Reset demo metrics
